Task 2: Configuring Renovate for dependency management

In this section, you will learn how to configure Renovate for dependency management. Follow the steps below to complete the task successfully.

1 Review the "Configure Renovate" Pull Request

↓

Once you have enabled Renovate on a repository, you will get a "Configure Renovate" PR (Pull Request) that looks something like this:

Now let's add some custom settings

2 Adding Labels and Reviewers

↓

In this step, you will learn how to add labels and reviewers to your Renovate configuration.

  • Open the renovate.json file in your repository.
  • To add labels, include a "labels" array in your configuration. For example: 
    
    {
      "labels": ["dependencies", "renovate"]
    }
  • To add reviewers, include a "reviewers" array with GitHub usernames. For example: 
    
    {
      "reviewers": ["team-member1", "team-member2"]
    }
  • Save the changes and commit the updated configuration file.

These settings will ensure that your pull requests are properly labeled and assigned to the right reviewers.

3 Configure Dependency Dashboard

↓

In this step, you will set up a Dependency Dashboard to track all dependency updates in one place.

  1. Open the renovate.json file in your repository.
  2. Add the following configuration to enable the Dependency Dashboard: 
    
        {
          "dependencyDashboard": true
        }
  3. Save the changes and commit the updated configuration file.

Once configured, Renovate will create a Dependency Dashboard issue in your repository to help you manage updates efficiently.

4 Dependency Dashboard

↓

In this step, you will have a look at the Dependency Dashboard to track all dependency updates.

  1. Click on the Github Issues
  2. There you should see a issue Named Dependency Dashboard
  3. Open the Dependency Dashboard issue to view all pending dependency updates.
  4. Review the list of updates and prioritize them based on your project's needs.
  5. Click on any update to view detailed information and take action, such as merging or closing the update.

5 Conclusion

↓

Congratulations! You have successfully configured Renovate for dependency management. You can now automate the process of keeping your dependencies up to date.
